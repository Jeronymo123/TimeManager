# TimeManager
Проект представляет собой Desktop-приложение, реализующее функции системы управления задачами.
Математические методы используются для учета времени и расчета разности временных меток между событиями входа и выхода из программы. 
Технология использует язык программирования C++, обеспечивающий высокую производительность. Для создания графического интерфейса используется фреймворк Qt.
